# Worked Examples — J-USE Pathway Engine

Three fictional Jamaica-realistic NbCS projects, one per pathway. Use these to train workshop participants, calibrate facilitators, or demonstrate the Pathway Engine end-to-end without referencing any real applicant.

| File | Pathway | Sector | Parish |
|---|---|---|---|
| `01-cane-valley-honey.md` | **Revenue Generating** | Apiculture + agroforestry + eco-tourism | St. Mary |
| `02-coastal-resilience-coop.md` | **Blended** | Mangrove restoration + community fisheries + hotel anchor | Hanover |
| `03-east-bay-greenway.md` | **Public Good** | Urban tree canopy + bioswales + parish-council adoption | Westmoreland |

## Important — these are fictional

Every organisation name, person, location detail, and number in these examples is **invented**. They are *not* drawn from any of the 48 actual J-USE 2026 EOI submissions. Use them freely in training without risk of bias toward or against real applicants.

## How to use

### As workshop hand-outs

Print or distribute the markdown file for the pathway you want to teach. Each example walks through every one of the 9 canvas blocks (Commercial / Stewardship-Adoption layer + Impact layer) plus the revenue model, sustainability test result, and example commitments.

### As Pathway Engine inputs

Workshop participants can be asked to **enter the worked example into the live Pathway Engine** as a hands-on exercise. They:
1. Open the Pathway Engine at the URL
2. Type the organisation name + reference number from the example
3. Pick the pathway it specifies
4. Copy each block's content into the form
5. See the sustainability test fire green / amber / red
6. Generate the landscape one-page PDF

This teaches them how the form works while seeing a fully completed canvas — better than starting blank.

### As facilitator calibration

Before running a participant workshop, facilitators can use these to:
- Confirm everyone has the same understanding of what "good" looks like for each block
- Test edge cases (e.g. what happens if the in-kind contribution is missing)
- Practise giving feedback on a known-good answer

## Style notes for the fictional organisations

- All organisations sit in **one of the 7 eligible J-USE parishes**
- Names blend Jamaican place-names with generic English ("Cane Valley", "East Bay", "Coastal Resilience") so they read as plausibly Jamaican without referencing real entities
- Numbers (beneficiary counts, JMD amounts, hectares) are realistic for the J-USE budget cap of JMD $15 million
- Each example is intentionally **a Tier 2 quality canvas** — solid but with one or two "fixable" gaps so participants can practise identifying improvements
